# OG Portfolio

A creative portfolio built with **Next.js (App Router)**. The site is styled with **Tailwind CSS** and adds subtle motion using **Framer Motion**.

## Requirements

- Node.js (LTS recommended)
- npm (or yarn/pnpm)

## Setup & Local Development

```bash
npm install
npm run dev
```

Then open:

- http://localhost:3000

## Build for Production

```bash
npm run build
npm run start
```

## Deploy (GitHub Pages)

This project uses `gh-pages` to publish the production build from the `out/` directory.

```bash
npm run deploy
```

## Tech Stack

- Next.js 16 + React
- TypeScript
- Tailwind CSS
- Framer Motion
